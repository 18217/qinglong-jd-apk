# 介绍
青龙脚本，京东cookie获取上传APP
- 手机使用app，无需电脑
- 手机使用app登录后，点击上传，会自动将cookie上传到青龙脚本服务器

# 使用方式
1. fork 本项目
2. 修改代码中的一些配置，如服务器地址，等
3. 在github右侧，创建一个release， 如v1.0.0。 等待一段时间就会构建出apk了
